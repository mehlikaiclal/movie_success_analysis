# Movie Success Analysis: A Business Perspective

## 📌 Overview

This project explores what makes a movie successful on a streaming platform by analyzing both quality (ratings) and popularity (number of ratings).

Instead of focusing on a single metric, this project introduces a simple framework to classify movies into meaningful categories.

---

## 🧠 Methodology

Movies are evaluated using two key dimensions:

* **Average Rating** → represents quality
* **Number of Ratings** → represents popularity

Based on these, movies are grouped into four categories:

* **Blockbusters** (high rating, high popularity)
* **Hidden Gems** (high rating, low popularity)
* **Overhyped** (low rating, high popularity)
* **Flops** (low rating, low popularity)

## 💡 Key Insights

* Hidden gems represent untapped opportunities for increasing user satisfaction
* Overhyped movies may reduce trust in recommendations
* Blockbusters drive engagement and can guide users toward other content

---

## 🎯 Business Implications

Platforms like Netflix can use this framework to:

* Promote high-quality but less visible content
* Improve recommendation strategies
* Make better content investment decisions

## 📌 Conclusion

This project shows how simple data analysis can support strategic decision-making in the entertainment industry by combining technical analysis with business thinking.
